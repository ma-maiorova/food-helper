# Парсер

Консольное приложение для парсинга данных с популярных сайтов доставки готовой еды

### Запуск

Сбор ссылок на все продукты готовой еды в файл ``links.csv``
```
main.py links --output="links.csv"
```

Парсинг собранных ссылок в `products.csv`

```
main.py parse --input="links.csv" --output="products.csv"
```

### Логика работы

У приложения две подпрограммы

#### `links`

Со страницы с готовой едой собирает ссылки на все странички с продуктами. Параметры:

- `--pages` - количество страниц для сайтов с пагинацией
- `--output` - файл, куда будет сохранен список ссылок (`csv`, `xlsx`)

#### `parse`

По собранным ранее ссылкам парсит каждую страницу с готовой едой. Параметры:

- `--input` - входной файл со списком ссылок (`csc`)
- `--output` - файл, куда будет сохранены собранные данные (`json`, `csv`, в будущем загрузка в БД)
- `--counts` - количество ссылок, которое будет обработано (если не хочется обрабатывать все)
